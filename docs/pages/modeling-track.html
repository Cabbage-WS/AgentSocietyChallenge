User Modeling Track

Introduction

This track focuses on simulating social interactions and information access behaviors on social media using open-source datasets. Participants will design and implement LLM-based agents to analyze historical user behavior records, model user preferences, and interact with the environment to simulate how users review businesses and engage with reviews. The goal is to enhance the design of effective information retrieval algorithms for web applications.

Participants are tasked with developing agents capable of understanding user preferences and behavior patterns by integrating user characteristics, item attributes, and other users' reviews. The resulting models should generate comprehensive judgments that include two key components for each review:

  • Star Rating – A numerical score reflecting the user's overall opinion.
  • Review Text – A detailed commentary providing specific feedback.

This track aims to explore the scalability and potential of LLM agents in simulating human behavior and their broader applications in behavioral modeling.

Evaluation

The performance of participants' models in simulating user interactions will be assessed through quantitative metrics, with a focus on accuracy in predicting user behavior. The evaluation criteria include:

  • Star Rating Accuracy
    • Metric: Mean Absolute Error (MAE)
    • Description: The predicted star ratings will be compared to ground truth values, normalized to the range [0,1].
    • Formula:
      \[ MAE = \frac{1}{N} \sum_{i=1}^{N} \left| \hat{s_{ni}} - s_{ni} \right| \]
  • Review Text Evaluation
    • Emotional Tone
      • Metric: MAE after normalization
    • Sentiment Attitude
      • Metric: MAE after normalization
    • Topic Relevance
      • Metric: Cosine distance between text embeddings
      • Formula:
        \[ d = 1 - \frac{u \cdot v}{\|u\|_2 \cdot \|v\|_2} \]
  • Final Error Score

    A weighted sum of the individual metrics for star rating accuracy, emotional tone, sentiment attitude, and topic relevance.
