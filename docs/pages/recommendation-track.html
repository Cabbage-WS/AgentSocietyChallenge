Recommendation Track

Introduction

This track focuses on developing LLM agents that function as recommendation assistants. Participants will design agents capable of performing recommendation tasks across various scenarios. Each task involves a specific user and a list of candidate items. The goal is to extract user preferences and item information from datasets to generate personalized recommendations.

The track aims to explore the role of LLM agents in advancing recommendation systems and next-generation information retrieval technologies, providing new perspectives on personalization and user engagement.

Evaluation

Participants' recommendation agents will be evaluated based on their ability to accurately rank items that users are most likely to select from a list of candidates. The ground truth for each task is generated by a predefined LLM agent simulating user choices based on context and task queries.

The evaluation criteria include:

  • Ranking Accuracy
    • Metric: Top-N Hit Rate (where \( N = \{1, 3, 5\} \))
    • Description: Measures how often the ground truth item appears in the top-N of the ranked list of 20 candidate items.
